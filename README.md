## 阶段总结

### 基础代谢：

13.7 * 体重公斤 + 5.0 * 身高厘米 - 6.8 * 年龄 + 66

13.7 * 65 + 5.0 * 177  - 6.8 * 24 + 66 = 1678.3kcal

再增加 300 ~ 500 kcal 创造热量盈余。

总摄入量 = 基础代谢 + 行为代谢 + 肌肉增长所需能量

运动代谢：基础代谢 * 运动系数 = 基础代谢 + 运动代谢

运动系数：每周运动 3 ~ 5 次 = 1.55

肌肉增长所需能量：上边的 300 ~ 500 kcal就是

折算下来，增肌所需热量 1678.3 * 1.55 +（300 ~ 500）= 2900 ~ 3100 kcal

1g 碳水 = 4 kcal

1g 蛋白 = 4 kcal

1g 脂肪 = 9 kcal



### 训练方式：

先复合，再单关节。

在掌握动作和发力的基础上，多做大重量复合关节，减少容量。大重量不是说离谱，就是多加一点就不行的那个度。

控制时间在 1.5 小时左右，最好控制在 1小时左右，绝对不能出现 2 个小时。
